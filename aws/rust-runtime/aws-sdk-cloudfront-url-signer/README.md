aws-sdk-cloudfront-url-signer
=============================

This crate is part of the [AWS SDK for Rust](https://awslabs.github.io/aws-sdk-rust/) and provides
utilities for CloudFront URL and cookie signing operations.

This crate provides functionality for:
- Signing CloudFront URLs with custom policies
- Creating signed cookies for CloudFront distributions
- Managing private keys for CloudFront signing
- Policy creation and validation
