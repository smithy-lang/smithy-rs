Add opt-in server codegen setting `rpcV2CborUseVerbatimOperationName` that keys the RPC v2 CBOR router on the verbatim Smithy operation shape name (per the smithy-rpcv2 spec) instead of the PascalCased Rust symbol name. Fixes client/server interop for operations whose names are not UpperCamelCase. Default is off to preserve existing behavior.
